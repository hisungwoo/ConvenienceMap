<h2>:memo: 상세 내용 </h1>

* 휠체어 사용자, 거동이 불편한 노인, 유모차 사용자처럼 이동과 출입이 불편한 교통 약자들을 위해 개발하였습니다.

* 내 위치를 기반으로 지도에 편의시설이 표시되며, 검색 서비스도 지원합니다.

* 공공데이터 포털의 편의시설 현황 API를 통해 편의시설 정보를 가져왔으며, 그 정보를 바탕으로 카카오맵 API를 사용한 지도에 마크를 표시합니다.

* 편의시설 정보의 좌표와 Google Street View API를 통해 시설 이미지를 가져옵니다.

* 팀 구성 : Android 개발자 1명

<br>
<h2>:computer: 개발 목표 </h1>

* 안드로이드 앱 전체 기획 및 개발

* 시스템 아키텍처 설계

* Jetpack 기능 활용하여 MVVM 패턴 적용

* Retrofit을 활용하여 API 연동

* 앱 출시 

* 출시 후에도 공부한 기술 스택 적용 후 지속적인 업데이트
